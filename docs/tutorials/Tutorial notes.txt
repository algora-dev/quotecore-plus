Catalogs:

- What
	- Change text to "Upload a CSV file containing columns and rows of information you want to display in a quote/order/invoice. No more digging through PDFs for price, description, item codes - Store it all here!"

- Why? (New page after "What")
	- Catalogs can help you save time writing or copying lines into quotes/orders/invoices. You can use anything from existing CSVs from suppliers, to fully custom made CSV files that hold all your quoting items from labor, materials, descriptions, pricing and more. This system is extremely modular and flexible allowing you to quote/order/invoice anything at all accurately and quickly while never having to leave or look away from the QuoteCore+ app

- Using it
	- Change text to " In a quote, invoice, or order: Add line → Search catalog → type → pick the item → it drops in with description + price. You can edit this completely after selection."

- Maps (New page)
	- You can add multiple column maps for each Catalog that you can search/pull from ie; this allows you to have 1 or 2 master Catalogs that give you all the data you need for quoting/ordering/invoicing but you might want to pull different info for each of those. Maps allow you to pull information that suits a quote, an order, or an invoice without needing to customise your CSV and upload 3 different CSVs.
You simply name the map, and then you can select that map when searching to line items.

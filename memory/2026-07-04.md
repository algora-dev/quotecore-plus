## Lessons (IMPORTANT — cost of this session's layout debacle)
- **Never diagnose JSX structure with regex div-counting.** Self-closing `<div ... />`
  (multi-line, e.g. color bars with style={{...}} />) count as opens with no close and
  corrupt every depth number after them. This produced 3 wrong "fixes" (flex-shrink-0,
  grid switch, calc height) before the real bug was found.
- **The browser DOM is ground truth.** `getComputedStyle` + `getBoundingClientRect` via
  browser evaluate found in 2 calls what hours of source reading missed.
- **Admin impersonation works for agent smoke-testing**: /admin/login (creds in MEMORY.md)
  → Users → user → Log in as → uncheck Notify → navigate to any workspace page. 30-min TTL.

# Creating material orders

Two ways to create an order: from scratch (custom), or by pulling from an existing quote.

## Custom orders

Create custom orders from scratch. You can add your own components or items, images or flashing drawings, quantities, and header/footer templates.

This system is designed to reduce the need to transfer orders from digital to pen and paper, significantly speeding up the process and resulting in a cleaner, more professional order form. The supplier ends up with something that has fewer mistakes and less risk of wrong items being delivered.

## Order from quote

The same as custom orders, but here you select an existing quote to become your order. When you select a quote, anything in your quote that you also allowed to be added to material orders will pre-populate the order form.

In some cases you may not need to change or edit the order form at all - just download and send to the supplier. But you can edit everything in the order if you need to:

- Flashings and images.
- Custom area-based components or items.
- Individual lengths and items.
- Header and footer.
- Layout changes from single to double column.
- Order of items.

## Setting up

For a component to appear in an order, the **Material orders** option needs to be enabled on the component itself. See [Creating a component](/docs/components/creating-a-component).

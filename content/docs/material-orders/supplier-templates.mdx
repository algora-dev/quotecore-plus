# Supplier templates

Supplier templates are the wrappers around a material order - the header (your details) and the footer (the supplier's details, delivery info, payment terms). Set them up once per supplier and pick from a dropdown when you build an order.

## When to use them

- You order from more than one supplier and they need different layouts.
- You want supplier addresses pre-filled instead of typed every order.
- You want consistent delivery and payment notes per supplier.

## Steps to create one

1. Go to **Material Orders** and find the supplier templates section.
2. Click **New Supplier Template**.
3. Set the name (the supplier's name is usually fine).
4. Build the header - your company info.
5. Build the footer - supplier address, contact, account number if you have one, payment terms, delivery notes.
6. Save.

## Steps to use one

When you build a [custom order](/docs/material-orders/custom-orders) or an [order from a quote](/docs/material-orders/order-from-quote), pick the template from the dropdown. The header and footer fill in. The line items are unaffected.

## What's in the template

- Header content - your company name, address, phone, contact.
- Footer content - supplier address, your account number, payment terms, delivery defaults.
- Default layout (single or double column).

## Tips

- Set up one template per supplier. Don't try to handle three suppliers with one template.
- If a supplier needs special instructions on every order (e.g. "deliver to depot, not site"), put those in the footer.
- Update templates when supplier addresses change.

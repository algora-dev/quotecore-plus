# Drawing flashings

The flashing drawing tool lets you build clean, re-usable flashing drawings you can attach to components and material orders. Draw it once, use it for every order that needs that flashing.

## When to use it

- You order custom flashings from a supplier and want a consistent drawing format.
- You want to stop re-drawing the same flashing on paper for every order.
- You want a way to attach a visual to a component so the supplier knows exactly what shape they're folding.

## Why it matters

Hand-drawn flashings on paper get misread. Smudged dimensions, ambiguous angles, "is that 75 or 175 mm?" - every roofer has had a delivery turn up wrong because of a sketch. The drawing tool produces consistent, dimensioned, easy-to-read drawings that suppliers can fold without guessing.

## Steps

1. Go to **Flashings** in the workspace nav.
2. Click **New Flashing**.
3. Use the drawing tools to lay out the profile - lines, angles, dimensions.
4. Add measurements where the supplier needs them.
5. Add custom values if the flashing has variable dimensions you'll set per quote.
6. Save with a clear name (e.g. "Apron flashing - 100x75 step").

## Attaching to components

Once you've saved a flashing drawing, you can attach it to a component. When you create or edit a component, there's a "Flashing drawing" field - pick the saved drawing.

When that component appears on a material order, the drawing goes with it.

## Printing and downloading

You can download or print any flashing drawing from the flashings page. Useful for paper records or when you need to show a fabricator something specific.

## Tips

- Use specific names. "Apron 100x75 step" beats "Apron".
- Keep variable dimensions to a minimum - one or two per drawing is fine, more gets confusing.
- If a flashing has a left-hand and right-hand version, make two drawings.

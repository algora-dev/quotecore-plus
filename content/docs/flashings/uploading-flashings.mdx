# Uploading flashings

If you've already got flashing drawings on file - hand-drawn, CAD exports, photos of profile samples - upload them and use them the same way as drawings made in the [flashing tool](/docs/flashings/drawing-flashings).

## When to use it

- You've got existing drawings you don't want to redraw.
- The flashing is a custom shape from a fabricator and you have a spec sheet.
- You'd rather use a photo than draw.
- You want a quick add for a one-off and you'll redraw it properly later if needed.

## Steps

1. Go to **Flashings** in the workspace nav.
2. Click **Upload Flashing**.
3. Pick an image from your computer (PNG or JPEG).
4. Name the flashing.
5. Save.

## Where uploaded flashings show up

Same place as drawn ones. Available to attach to components, available to drop into material orders.

## Tips

- Use clear, square-on photos. Side-on shots are hard for suppliers to interpret.
- Keep file sizes reasonable - under 1 MB is plenty.
- Crop the image to the flashing only. Don't include the whole worktop.

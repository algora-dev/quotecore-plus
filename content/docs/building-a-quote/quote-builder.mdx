# Quote builder

The quote builder is where you define exactly what shows up in your quote: materials, labor, components, areas, items, and extras.

## The four phases

### 1. Areas

Add or edit areas (main, garage, lean-to, dormer, etc.) and their names, the pitch if applicable, and the actual area measurements.

### 2. Components

Add or edit all aspects of the components you want in the quote: which areas they apply to, individual lengths, units, and items. Each component can be flipped between [Plan and Actual](/docs/concepts/plan-vs-actual) here.

### 3. Extras

Add or edit any extras: one-off items, multipliers, delivery, accommodation, skip hire, anything outside the main components. See [Components vs Extras](/docs/concepts/components-vs-extras).

### 4. Review

The summary view. You can see the totals, set a profit margin percentage for materials and labor, and save.

## Editing later

You can go back and edit any of the quote builder phases once you have saved. You can also manually add items you missed later in the customer quote builder.

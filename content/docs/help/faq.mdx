# FAQ

The questions we get most often, with quick answers.

## What's the difference between a Component and an Extra?

Components are the materials and labor that make up the core job (tiles, longrun, ridges, fixings). Extras are everything else - skip hire, accommodation, scaffold, custom one-offs. Both behave the same for pricing; the split is about how the quote reads. See [Components vs Extras](/docs/concepts/components-vs-extras).

## Can I edit a sent quote?

Yes. Either:

- Edit the customer quote directly (small text changes) in the [customer quote editor](/docs/customer-facing/customer-quote-editor) and save.
- Edit the underlying quote (numbers and components) by going back into the [quote builder](/docs/building-a-quote/quote-builder) from the summary.

Generate a fresh URL afterwards so the customer sees the latest version. The old URL becomes invalid.

## How do I handle a re-roof where I already have actual measurements?

Use [Manual Quote](/docs/building-a-quote/manual-quote). Add your components, type in the measurements, and set each one to **Actual** so the pitch maths are skipped. Waste still applies. See [Plan vs Actual](/docs/concepts/plan-vs-actual).

## Can the customer see my labor cost?

Not unless you specifically choose to. The [labor sheet](/docs/labor-and-installers/labor-sheet-editor) is separate from the customer quote. Labor amounts on the customer quote are bundled into the line price by default.

## How do I clone a quote?

From the [quote summary](/docs/building-a-quote/quote-summary), click **Clone Quote**. Enter a new customer name. Edit anything that needs changing.

Cloning is faster than starting fresh when:

- You are quoting a similar job.
- You messed up a quote and want a clean restart.
- You are quoting the same job for a different customer.

## What measurement systems are supported?

Three: Metric (m2, m, mm), Imperial feet (ft2, ft, inches), and Imperial roofing squares (Rs, linear ft). Pick a default in [Company settings](/docs/account/company-settings). Each new quote locks to its system at creation; existing quotes don't change if you flip the default later.

## How do I add a one-off cost like skip hire?

Add it as an **Extra**. Quickest way: add it as a line item directly in the quote builder Extras phase. If it is something you use on multiple quotes, save it as a component with type "Extra" so it lives in your library.

## How do I back up everything?

From the [quotes page](/docs/building-a-quote/quote-summary), check the box next to one or many quotes (up to 25 at a time) and click **Download**. You get a zip containing the quotes, customer quotes, labor sheets, and all attached files. Drop it on your computer, a USB stick, or your cloud drive.

## How do I delete a quote?

Same place - check the box, click **Delete**. Limit is 25 quotes per batch. Deleted quotes are gone permanently; download first if you might want them back.

If you might want to reference the quote later but it should not be valid for the customer, [withdraw](/docs/customer-facing/withdrawing-a-quote) instead.

# Changelog

Recent updates to QuoteCore+.

## May 2026

- New help library at `/docs` with full coverage of every feature.
- In-app help drawer (the question-mark icon top right) opens to the doc page for whatever you're looking at.
- Search across docs from the top of the help library.

## Earlier 2026

- Multi-tax support per quote with separate quote and labor sheet visibility.
- Customer quote and labor sheet templates with default selection.
- Quote URL system with accept, decline, and request-changes buttons.
- Withdraw quote with re-quote request capture from the expired link page.
- Bulk download and delete from the quotes page.
- Account, security, notifications split across separate pages.

## Coming soon

- [Team members](/docs/account-and-billing/team-members).
- [Follow-up emails](/docs/customer-facing/follow-up-emails).
- Project Manager mode (schedules, dates, custom notes per accepted job).

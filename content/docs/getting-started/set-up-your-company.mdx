# Set up your company

Your company settings drive what shows up on quotes, what currency things are priced in, and which measurement system new quotes start in. Spend five minutes here before you build a quote and you won't have to think about it again.

## When to use it

- Right after you sign up.
- Whenever you change your logo, address, or contact details.
- If you're starting to quote in a different currency or unit system.

## Steps

1. Go to **Account > Company**.
2. Add your company name (this overrides the one you set at sign-up if it's different).
3. Upload your logo. PNG with a transparent background works best - it lands on every customer quote.
4. Set your default currency.
5. Set your default measurement system: metric, imperial feet, or imperial roofing squares.
6. Set your default profit margin for materials and labor. This applies to new quotes; you can override it on each quote.
7. Add your tax rates under the Taxes panel. You can have one or several. Each tax has a name, a rate, and toggles for whether it shows on customer quotes and labor sheets.
8. Save.

## Field reference

<Field name="Company name" required>
The name your customers see at the top of every customer quote and on the email "from" line. Edit it any time.
</Field>

<Field name="Logo">
Upload from your computer. Use a transparent PNG if you can. Customer quotes show it in the top right of the document.
</Field>

<Field name="Default currency" required>
Three-letter currency code (NZD, AUD, USD, GBP, and so on). New quotes start in this currency. Existing quotes keep the currency they were created in.
</Field>

<Field name="Default measurement system" required>
Metric (meters, m2), imperial feet (ft, ft2), or imperial roofing squares (Rs). Each new quote locks to the system you chose at creation - you can change the default for future quotes any time.
</Field>

<Field name="Default profit margin">
A percentage applied to materials and labor when you build a new quote. Set both to zero if you'd rather mark up by hand on each quote.
</Field>

<Field name="Taxes">
A list of tax rates - GST, VAT, or anything else you charge. Each row has a name (shown to the customer), a rate, and a toggle for whether it appears on the customer quote and the labor sheet. New quotes copy this list when they're created, so editing it later doesn't change quotes you've already sent.
</Field>

<Callout type="warning">
Changing a tax rate later only affects new quotes. If you need to update a quote you've already created, edit the tax inside that quote.
</Callout>

## Tips

- If you work in a region with multiple taxes (provincial + federal, for example), add each one as its own row. The customer quote shows them as separate lines.
- Logo files load on every customer quote PDF. Keep the file under 500KB so PDFs render fast.

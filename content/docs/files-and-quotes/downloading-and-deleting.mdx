# Downloading and deleting quotes

Both actions live on the **Quotes** page, behind a checkbox-select on each row.

## Download quotes

You can download your quote and any uploaded files at any time. From the quotes page, check the box to the left of your chosen quote(s) and a download/delete option appears. Click **Download** to receive the quote in a clean zip file containing all the information your quote was derived from.

The zip contains:

- The quote summary.
- The customer quote (if you created one).
- The labor sheet (if you created one).
- All attached files (plans, photos, supporting docs).
- The takeoff image, if you used digital takeoff.

## Delete quotes

Same flow as downloading. Check the box next to the quote(s) you want to delete, and click **Delete**.

<Callout type="warning">
Once you delete a quote, it is gone forever. We recommend downloading first as a backup. It is always better to have them.
</Callout>

## Batch limits

You can select up to **25 quotes at a time** for download or delete. If you have more, process the first batch and then select the next 25.

This cap exists to protect the system: very large batches can time out, run into browser memory limits, or leave the database in a half-applied state.

## Withdrawing instead of deleting

If a quote was sent but never accepted, and you might want to reference or re-send it later, [withdraw](/docs/customer-facing/withdrawing-a-quote) it instead of deleting. The quote stays in your account but its URL tells the customer it is no longer valid.
